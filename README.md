# 📍 Yandex Maps Reviews Integrator

Инструмент для автоматического сбора и отображения отзывов из Яндекс Карт на стеке **Laravel 11 + Vue 3**.

---

## 📖 Обзор системы
Проект позволяет импортировать отзывы в локальную базу данных. Это обеспечивает мгновенную загрузку страницы и возможность работы с данными даже без прямого доступа к API Яндекса.

**Архитектура:** `Yandex Maps` ➔ `Apify Scraper` ➔ `Laravel DB` ➔ `Vue UI`



---

## 🛠 Что нужно для старта (Pre-requisites)
Перед установкой убедитесь, что у вас есть:
* **PHP 8.2+** и **Node.js 18+**.
* **Аккаунт на [Apify.com](https://apify.com/)**:
    1. Зайдите в `Settings` -> `Integrations`.
    2. Скопируйте **Personal API token**.
    * *Это необходимо, так как парсинг идет через их облако.*

---

## 🚀 Быстрая установка (Installation)

### 1. Клонирование
```bash
git clone https://github.com/hezam17/yandex-maps-integration.git
cd yandex-maps-integration
```
### 2. Зависимости и окружение
```bash
composer install && npm install
cp .env.example .env
php artisan key:generate
```


### 3. Настройка .env
Укажите данные вашей БД и токен Apify:
```bash
APIFY_API_TOKEN=ваш_токен_из_шага_выше
APIFY_ACTOR_ID=zen-studio~yandex-maps-scraper
```
### 4. 4. Запуск базы и сервера
```bash
php artisan migrate
npm run dev
# В другом окне:
php artisan serve
```
# ⚙️ Логика работы API (How it works)
Для тех, кто хочет понять, как данные попадают в БД:

Запуск (Start): Laravel отправляет POST-запрос к Apify с URL компании.

Опрос (Polling): Система каждые 5 секунд проверяет статус задачи в облаке Apify.

Загрузка (Download): Как только парсинг завершен, Laravel скачивает JSON с отзывами.

Сохранение (Upsert): Данные нормализуются и сохраняются в БД. Если отзыв уже есть, он обновляется (без дубликатов).

🔄 Инструкция по использованию
Откройте приложение, перейдите в Settings.

Вставьте ссылку на отзывы организации (например: https://yandex.ru/maps/org/.../reviews/).

Нажмите Save. Система автоматически запустит первый сбор данных.

Перейдите в раздел Reviews для просмотра.

💡 Важные заметки
🛡 Безопасность: Ваш токен никогда не попадает в браузер, все запросы идут через бэкенд.

💰 Экономия: Данные кэшируются на 6 часов. Частые обновления не будут тратить ваш баланс в Apify.
